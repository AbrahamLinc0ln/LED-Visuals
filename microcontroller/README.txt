A place for different visual effects implemented using a microcontroller and an addressable RGB LED strip (ws2128b). Requires the Adafruit Neopixel library. Below are descriptions of the different visual effects.

comet: slides across strip with a randomly decaying tail
comet_flavor: same as comet but with randomness
akira_pill: various visual effects for a matrix of LEDs in the shape of the pill from akira
